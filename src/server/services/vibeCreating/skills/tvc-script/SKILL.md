# TVC 剧本创作

## 核心职责
基于用户产品图和需求分析信息，创作专业的TVC广告剧本大纲。

## 输入参数
- **产品图**：从步骤0获取的产品图片
- **品牌定位**：从步骤0获取的品牌认知和愿景
- **目标客户**：从步骤0获取的目标客户信息和特征
- **美学理念**：从步骤0获取的视觉风格偏好
- **品牌使命**：从步骤0获取的品牌使命
- **核心信息**：从步骤0获取的核心信息
- **广告目的**：从步骤0获取的广告明确目的
- **时长要求**：TVC 时长（秒）
- **目标受众群体**：从步骤0获取的明确目标受众

## 执行步骤

### 1. 产品分析
分析产品图，确定：
- 产品外观特征（颜色、形状、设计风格）
- 产品功能特性（核心卖点）
- 产品使用场景（适用人群、使用环境）
- 产品情感属性（高端、亲民、科技感等）

### 2. 品牌定位结合
结合步骤0的需求分析：
- 将品牌愿景融入剧本主题
- 根据目标受众确定故事风格
- 根据美学理念确定视觉风格
- 根据品牌使命确定情感基调
- 根据核心信息确定传播重点
- 根据广告目的确定剧本结构

### 3. 剧本创作
基于产品分析和需求分析，创作剧本大纲，包含：
- 广告主题（符合品牌愿景）
- 创意概念（针对目标受众）
- 故事线概述（体现品牌使命）
- 核心传播点（传递核心信息）
- 情感基调（符合美学理念）
- 产品展示方式（自然融入剧情）

### 4. 目的导向
根据广告目的优化剧本：
- **吸引客户留资**：强化产品价值，引导行动
- **建立品牌形象**：强调品牌调性，传递情感
- **实现产品销售**：突出产品卖点，激发购买欲
- **提升品牌认知**：创造记忆点，增加曝光

## 注意事项
- 剧本要符合TVC时长要求（通常15-60秒）
- 产品展示要自然融入剧情，避免硬广感
- 核心传播点要清晰明确
- 情感基调要与品牌定位一致
- 必须基于步骤0收集的需求分析信息创作
- 剧本要服务于明确的广告目的
- 通用HTML标签（<step>、<title>、<content>、<response>、<item>、<fields>、<field>）由系统提示词定义
- 本步骤专用标签（<sections>）在技能文件中定义
